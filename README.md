# BeyondBites

## Overview

BeyondBites is an Android application that helps restaurants reduce food waste by selling surplus food at discounted prices.

## Technologies Used

- Kotlin
- Firebase Authentication
- Firebase Firestore
- Android Studio

## Features

- User Registration
- Login Authentication
- Restaurant Dashboard
- Food Listing
- Volunteer Management
- Real-Time Updates


## My Role

- Developed Android application
- Designed Firebase database
- Implemented user authentication
- Built volunteer coordination module
